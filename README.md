# stockInCommandLine

在命令行使用python实时查看股市信息


====================================

#2015-05-30

1.	显示股票的的实时数据（昨收、今开、最高、最低、现价）
2.	显示相对于昨收价的涨跌颜色（昨收：白色；今开、最高、最低、现价、涨跌：红涨绿跌）
